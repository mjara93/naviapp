== NAVIAPP

Sistema de administración y seguimiento de embarcaciones vía GPS.
Desarrollada en Ruby on Rails 4, cuenta con una API para la conexión de una aplicación android para el posicionamiento de embarcaciones vía GPS, la cual obtiene la posición y la envía en formato JSON a la aplicación.

* Aplicación alojada en Heroku
	https://naviera.herokuapp.com
